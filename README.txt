
SISTEMA DE PRODUTIVIDADE - VERSÃO COMPLETA

Como usar:
1. Instale os requisitos:
   pip install flask bcrypt pandas openpyxl

2. Execute o sistema:
   python app.py

3. Acesse no navegador:
   http://localhost:5000

Usuário de teste:
- Funcionário:
    Matrícula: 123
    Senha: 1234

- RH:
    Usuário: admin
    Senha: admin123

Funcionário pode:
- Ver sua produtividade, estoque, bônus
- Ver e concluir tarefas
- Solicitar uso de pontos do estoque
- Ver histórico e gráficos

RH pode:
- Cadastrar e editar funcionários
- Cadastrar e gerenciar tarefas
- Ver solicitações de resgate
- Filtrar dados e exportar relatórios
